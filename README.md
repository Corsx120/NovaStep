# 🚀 NovaStep — Программный комплекс «Менеджер задач»

**NovaStep** — это современная кроссплатформенная система для управления личной эффективностью, включающая Desktop-версию (Windows) и мобильное приложение (Android). Проект разработан как итоговая работа, демонстрирующая владение многослойной архитектурой, паттернами проектирования и интеграцией внешних сервисов.

---

## 🎯 Назначение и цели
Основная цель системы — снижение когнитивной нагрузки на пользователя через структурирование дел, приоритизацию и визуализацию прогресса. 

**Ключевые возможности:**
- **Умное управление задачами:** Полноценный CRUD-цикл с поддержкой дедлайнов, групп и тегов.
- **Голосовой ввод:** Интеграция технологии *Speech-to-Text* для создания заметок «на лету» без использования клавиатуры.
- **Интеллектуальная статистика:** Визуализация продуктивности и настроения за неделю, месяц и год с использованием динамических графиков.
- **Дневник состояния:** Встроенный трекер настроения и готовности к работе для анализа эмоционального фона.
- **Безопасность:** Защита персональных данных при помощи PIN-кода на входе.
- **Резервное копирование:** Экспорт и импорт данных в универсальном формате JSON для предотвращения потери информации.

---

## 🛠 Технологический стек
Приложение построено на современном стеке технологий, обеспечивающем высокую производительность и единую логику для разных платформ:

* **Язык программирования:** Dart
* **Фреймворк:** Flutter (Desktop & Mobile)
* **Управление состоянием (State Management):** Provider (Паттерн Observer)
* **База данных:** SQLite (локальное хранилище в 3НФ)
* **Архитектурный паттерн:** Singleton (для работы с БД), Observer, Layered Architecture (UI, Logic, Data)

## ⚙️ Инструкция по запуску

### Предварительные требования
Для сборки проекта необходимо наличие:
1. Flutter SDK (версия 3.10.7 или выше).
2. Android Studio (для Android сборки).
3. Visual Studio с поддержкой C++ (для сборки под Windows).

### Запуск приложения
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Corsx120/NovaStep.git
   ```
2. Перейдите в папку приложения:
   ```bash
   cd app
   ```
3. Установите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение:
   - **Windows:** `flutter run -d windows`
   - **Android:** `flutter run -d android`
